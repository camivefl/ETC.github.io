Lista de Cursos

  • Introducción a Excel


    Este curso está diseñado para brindarte los conocimientos fundamentales y las habilidades
    necesarias para utilizar Excel de manera eficiente y efectiva. Excel es una poderosa
    herramienta de hoja de cálculo utilizada en empresas, organizaciones y en tu vida cotidiana
    para realizar tareas numéricas, organizar datos y realizar análisis.

  • Introducción a Word


    Este curso está diseñado para brindarte los conocimientos esenciales y las habilidades
    necesarias para utilizar Microsoft Word de manera efectiva. Word es una herramienta de
    procesamiento de textos ampliamente utilizada en entornos académicos, profesionales y
    personales para crear documentos, informes, cartas y más.

  • Navegación segura


    En este curso, aprenderás las mejores prácticas y técnicas para navegar por Internet
    de manera segura y proteger tu información personal. A medida que el mundo digital
    evoluciona, es crucial tener conocimientos sólidos sobre la seguridad en línea para
    evitar riesgos y proteger tus datos confidenciales.
